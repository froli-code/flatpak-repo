# Frolicode Flatpak Repository
This is the official Flatpak repository for apps by frolicode

## Adding this remote
Use the following command to add this repo: `flatpak remote-add --if-not-exists myrepo https://github.com/froli-code/flatpak-repo/frolicode.flatpakrepo`

## Installing individual apps
To install an app, visit the respective project page for installation instructions.
